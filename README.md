# Matech_Solutions_chatbot_n8n
Chatbot orquestrado no n8n utilizando a arquitetura RAG. O sistema converte consultas em vetores, recupera trechos de documentos de uma base de conhecimento específica (DB Vetorial) e usa esse contexto para aumentar o prompt enviado ao LLM. Isso garante respostas precisas e fundamentadas no domínio do projeto.
